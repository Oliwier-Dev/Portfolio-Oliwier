How to use this AI chat

This assistant is designed for quick question-and-answer support based on the local knowledge file for this project.

How to use it

  1. Type a clear question in the input bar at the bottom.
  2. Press Enter or click the send button.
  3. Your message appears on the right, and the assistant answer appears on the left.
  4. Use specific terms related to your topic to get a stronger match.

How it works

The script loads data from kb.JSON. Each knowledge item has keywords, a question label, and an answer. Your input is converted to lowercase and compared against each keyword. Every match adds a score. The highest-scoring item is selected and rendered as the response.

Best results: ask focused questions and include key terms that are likely to exist in the knowledge base.

Examples

Try to include keywords such as: website cost, start business, planets solar system, side hustle 2026, go viral or productivity apps
